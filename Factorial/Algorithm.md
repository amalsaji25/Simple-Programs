## ALGORITHM - FACTORIAL
----------------------------------------------------------------
1. START
2. READ NUMBER FROM USER, STORE IN NUM
3. INVOKE FUNCTION FACT
4. DISPLAY FACTORIAL OF NUMBER
5. STOP

## SUB PROGRAM - FACT

1. START
2. IF NUM >= 1
    2.1 RETURN NUM * FACT(NUM - 1)
3.  ELSE
    3.1 RETURN 1
4.  STOP
